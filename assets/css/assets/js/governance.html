Governance Core

Die Kette

Policy → Identity → Audit → Execute. Dazu: SafeMode, Timeboxing, Target-Scoping, Explainability.

Policy

Allow/Deny + Zeitfenster + Profile. Kein Bypass.

Identity

Owner Unlock / Biometrie / Rollen (später org-fähig).

Audit

Immutable Event-Trail, exportierbar (PDF/JSON).

Execute

Nur freigegebene Actions, scoped & timeboxed.
